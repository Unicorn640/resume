# Resume Website

This repository contains a simple static resume website built with HTML and CSS.

## Files

- `index.html` — main resume page
- `styles.css` — page styling

## How to use

1. Open `index.html` in your browser.
2. Edit the sample text in `index.html` to include your name, experience, education, and projects.
3. Update `styles.css` if you want to adjust colors, spacing, or layout.
